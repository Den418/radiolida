# 📻 RadioPlayerV3

Бот для воспроизведения музыки и интернет-радио в голосовых чатах Telegram.

---

## ⚡ Что исправлено в этой версии

| Проблема в оригинале | Исправление |
|---|---|
| `bot.run(main())` блокирует процесс — весь код ниже никогда не выполняется | Весь код перенесён в `async def main()` |
| `Thread(target=func())` — вызывает функцию сразу, а не передаёт её | Исправлено на `Thread(target=func)` без скобок |
| Два разных клиента бота (`utils.py` + `main.py`) с одним токеном | Единый клиент в `bot_client.py` |
| `RADIO = {6}` — нелогичная инициализация | Состояние хранится через `playlist` (пусто = радио) |
| Хардкод чужого admin ID (`1316963576`) в трёх файлах | Удалён, используется только `AUTH_USERS` |
| Устаревший API PyTgCalls (`GroupCallFactory`) | Обновлено до PyTgCalls >= 0.9 (`PyTgCalls`, `MediaStream`) |
| `FloodWait.x` — атрибут переименован в новом Pyrogram | Исправлено на `FloodWait.value` |
| `ydl.extract_info()` вызывается при импорте `config.py` | Убрано из импорта, вызывается лениво |

## 🛠 Требования

- Python 3.10+
- FFmpeg (для конвертации аудио)
- Отдельный аккаунт Telegram для юзербота (не основной!)

## 🚀 Установка и запуск

### 1. Клонируй репозиторий
```bash
git clone https://github.com/AsmSafone/RadioPlayerV3
cd RadioPlayerV3
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

Установи FFmpeg (если не установлен):
```bash
# Ubuntu/Debian
sudo apt install ffmpeg

# macOS
brew install ffmpeg

# Windows — скачай с https://ffmpeg.org/download.html
```

### 3. Получи строку сессии юзербота
```bash
python3 gen_session.py
```
Следуй инструкциям. Скопируй SESSION_STRING.

### 4. Настрой переменные окружения
```bash
cp .env.example .env
nano .env   # или открой в любом редакторе
```

Заполни обязательные поля (см. раздел «Настройка» ниже).

### 5. Запусти бота
```bash
python3 main.py
```

## ⚙️ Настройка

### Обязательные переменные

| Переменная | Как получить |
|---|---|
| `API_ID` | https://my.telegram.org → API development tools |
| `API_HASH` | там же |
| `BOT_TOKEN` | @BotFather → /newbot |
| `SESSION_STRING` | `python3 gen_session.py` |
| `CHAT_ID` | ID канала/группы (отрицательное число) |

### Как узнать CHAT_ID
1. Перешли любое сообщение из нужного канала боту @username_to_id_bot
2. Или добавь @JsonDumpBot в группу — он покажет ID

### Необязательные переменные

| Переменная | По умолчанию | Описание |
|---|---|---|
| `AUTH_USERS` | пусто | Дополнительные администраторы (ID через пробел) |
| `STREAM_URL` | тестовое радио | URL радио-потока (.mp3, .aac, .m3u8...) |
| `LOG_GROUP` | пусто | ID группы для логов |
| `ADMIN_ONLY` | False | Только админы могут добавлять треки |
| `DELAY` | 10 | Секунд до удаления служебных сообщений |
| `EDIT_TITLE` | True | Менять заголовок голосового чата |
| `RADIO_TITLE` | `📻 РАДИО 24/7` | Заголовок при работе радио |
| `MAXIMUM_DURATION` | 15 | Лимит длины трека в минутах |

## 📋 Команды

### Музыка
| Команда | Описание |
|---|---|
| `/play [ссылка/название]` | Добавить трек из YouTube в очередь |
| `/song [ссылка/название]` | Скачать аудио и отправить файлом |
| `/skip` | Пропустить текущий трек |
| `/pause` | Пауза |
| `/resume` | Продолжить |
| `/replay` | Повторить с начала |
| `/current` | Что сейчас играет |
| `/playlist` | Список очереди |

### Радио
| Команда | Описание |
|---|---|
| `/radio` | Включить радио (из STREAM_URL) |
| `/stopradio` | Остановить воспроизведение |

### Голосовой чат (только администраторы)
| Команда | Описание |
|---|---|
| `/join` | Войти в голосовой чат |
| `/leave` | Выйти из голосового чата |
| `/mute` | Заглушить юзербота |
| `/unmute` | Включить звук |
| `/volume 0-200` | Установить громкость (100 = стандарт) |

### Обслуживание (только администраторы)
| Команда | Описание |
|---|---|
| `/clean` | Удалить загруженные файлы |
| `/restart` | Обновить код и перезапустить |
| `/setvar КЛЮЧ ЗНАЧЕНИЕ` | Изменить Config Var на Heroku |

## 🏗 Деплой на Heroku

1. Создай приложение на https://heroku.com
2. Подключи GitHub-репозиторий
3. В **Settings → Config Vars** добавь все переменные из `.env`
4. В **Resources** включи dyno `worker`

## ❓ Частые проблемы

**«Не удалось войти в голосовой чат»**
→ Убедись, что юзербот является участником канала/группы
→ Создай голосовой чат вручную один раз — бот создаст его автоматически при следующем запуске

**«FloodWait»**
→ Telegram временно ограничил запросы. Бот подождёт и продолжит автоматически.

**«PyTgCalls: GroupCallFactory not found»**
→ Обнови библиотеку: `pip install -U pytgcalls`

**Трек с YouTube не воспроизводится**
→ Обнови yt-dlp: `pip install -U yt-dlp`
→ YouTube иногда меняет API, yt-dlp быстро обновляется
